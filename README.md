
  # Premium Online Store Website

This is a code bundle for Premium Online Store Website. The original project is available at https://www.figma.com/design/f749fUPtyAKtWMkcpslg8X/Premium-Online-Store-Website.

## Prerequisites

- Node.js 18 или выше (npm идёт в комплекте с Node.js)
- Visual Studio Code

## Recommended VS Code extensions

- ESLint
- Prettier
- Tailwind CSS IntelliSense
- npm Intellisense
- GitLens (опционально)

## Установка зависимостей

Откройте проект в VS Code и выполните в терминале:

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

После запуска откройте в браузере адрес, который покажет Vite (обычно `http://localhost:5173`).

## Сборка проекта

```bash
npm run build
```

Если нужно, можно посмотреть созданную папку `dist/` после билда.
